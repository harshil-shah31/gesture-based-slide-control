# Gesture Based Slide Control

This project implements a touchless presentation control system using computer vision.

The system uses a webcam to detect hand gestures and control slides in real time.

Features:
- Next slide
- Previous slide
- Pointer display
- Draw annotations
- Erase annotations
- Exit presentation

Technologies Used:
- Python
- OpenCV
- cvzone
- Mediapipe
- NumPy
